ProxySettings
=============

Параметры настройки прокси-сервера


.. rubric:: Свойства

:Url:
    **Строка, чтение/запись** - URL прокси-сервера

:Login:
    **Строка, чтение/запись** - логин учетной записи прокси-сервера

:Password:
    **Строка, чтение/запись** - пароль учетной записи прокси-сервера
